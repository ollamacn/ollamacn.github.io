轻松在运行大型语言模型

上手使用 Deepseek, Qwen3,Gemma3 等模型，支持 macOS, Linux, 和 Windows。

下载 macOS 版 下载 Windows 版
~ ollama run qwen3
>>> 这是一个本地运行的模型！

在 Linux 上安装

通过一行命令即可快速部署。此脚本已针对国内环境优化。

curl -fsSL https://ollama.quickso.cn/install.sh | sh

为什么选择 Ollama？

  • 隐私保护：所有数据均在本地处理
  • 极致轻量：安装包极小，资源占用低
  • 丰富的模型库：一键下载主流开源模型
